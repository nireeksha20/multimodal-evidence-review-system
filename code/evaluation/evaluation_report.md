# System Capabilities

- Claim extraction from customer-support conversations
- Multi-image evidence review using Gemini Vision
- Visible damage detection
- Object part identification
- Evidence sufficiency assessment
- Claim support / contradiction / insufficient information decisions
- Severity estimation
- User-history risk assessment
- Prompt-injection resistance
- Structured CSV output generation

## Known Limitations

- Some complex conversational claims may reduce extraction accuracy.
- Multi-damage claims may require manual review.
- Results depend on image quality and model confidence.
